About Me

My Journey & Reflections

Who I Am

I’m Zariyat Adeniran, a curious and driven Full Stack Developer from Nigeria. I love building things that make life easier, whether it’s a clean user interface or a backend system that just works. My journey into tech started with a simple desire to understand how websites function, and it’s grown into a full-blown passion for solving problems through code. I enjoy learning, experimenting, and sharing what I know with others. For me, tech isn’t just about tools, it’s about impact.

My Goals in This Program

  • Sharpen my frontend skills, especially with React and modern JavaScript
  • Build real-world projects that showcase my growth and creativity
  • Learn how to write clean, maintainable code
  • Get comfortable working in teams and contributing to open-source
  • Push myself outside my comfort zone and grow as a developer
  • Build confidence in deploying and maintaining web applications

Areas Where I Want to Grow

  • Advanced JavaScript concepts like closures and async/await
  • Writing efficient backend logic and understanding system architecture
  • Debugging complex issues and optimizing performance
  • Navigating Git workflows with confidence
  • Communicating technical ideas clearly in team settings
  • Writing tests and thinking more about code reliability
